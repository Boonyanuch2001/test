    การ Start service sql server

      การ Start service sql server
      1. เปิด Services Manager:
          a. กด Win + R เพื่อเปิดหน้าต่าง Run.
          b. พิมพ์คำว่า services.msc แล้วกด Enter.
      2. ค้นหา Service ที่ชื่อว่า SQL Server (MSSQLSERVER):
          a. ในหน้าต่าง Services, ค้นหารายการชื่อ SQL Server (MSSQLSERVER) ในรายการ.
      3. Start Service:
          a. คลิกขวาที่ Service นั้น แล้วเลือก Start.
          b. รอให้สถานะของ Service เปลี่ยนเป็น Running.
