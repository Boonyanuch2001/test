    การตั้งค่า ODBC

    1. ติดตั้ง SQL Server Native Client Driver


    2. ตรวจสอบ enable protocol TLS 1.2 โดยไปที่Registry Editor

      Path : \HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\SecurityProviders\SCHANNEL\Protocols


    3. Add ODBC (32-bit) โดยจะต้องใช้สิทธิ์ User Admin ในการ Add ODBC (32-bit) แบบ SQL Server Native Client และ Test Connection Database


    4. ไปยัง Tab System DNS และ คลิก Add..


    5. ทำการเลือก Driver SQL Server Native Client


    6. ทำการะบุ ชื่อ ODBC Name และระบุ Database Server แล้วคลิก Next >


    7. ระบุ User Name และ Password สำหรับเข้าใช้งาน Database จากนั้น คลิก Next >


    8. เลือก Database ที่ต้องการจะเชื่อมต่อ


    9. กดปุ่ม Finish เพื่อทำการสร้าง ODBC Data Source


    10. ทำการทดสอบ Connect Database โดย และคลิก Test Data Source…
