ย้อนกลับ

    ทดสอบการใช้งานระบบ

    1. ทดสอบการใช้งาน Bonanza Investment Operation
        a. เข้าไปที่ Server Path\Operation คลิกขวา Operation.exe แล้วเลือก Run as Administrator


        b. ทำการทดสอบ Log in ด้วย User ของระบบ Bonanza ที่เตรียมไว้ให้


        ต้องสามารถเข้าสู่ระบบได้ แล้วกด exit เพื่อปิด application


    2. ทดสอบการใช้งาน Fluent UI
        a. ทดสอบเข้าใช้งานจากโปรแกรม Bonanza Investment Operation.exe เลือกเมนู Interface -> Extension


        b. ต้องสามารถเปิดหน้าจอเข้าใช้งาน Fluent UI ได้
