    การกำหนดค่า ENVSetup.EXE

      แนะนำ สามารถดูเครื่อง Client อื่นเพื่อเป็นตัวอย่างในการกรอกค่าได้

      1. คลิกขวา Run as administrator ที่ ENVSetup.exe - Shortcut กำหนดค่าดังต่อไปนี้
      2. Local Path = กำหนด Path ของผู้ใช้ที่สามารถ read และ write ได้ (ควรเป็น Path บนเครื่อง Client) ตัวอย่าง
        เช่น C:\Bonanza ตามข้อ 1 สามารถสร้าง Folder ย่อยเพื่อเก็บไฟล์แยกจาก Shortcut ได้
      3. Server Path = path ไปยัง Application Server ของ Bonanza ตัวอย่างเช่น X:\
      4. Database = กำหนดประเภทของฐานข้อมูล SQL Server
      5. DB Server = กำหนด Server Name หรือ IP ของ Server
      6. DB Name = กำหนด Database Name บน Server
      7. Server Name = กำหนด Server Name หรือ IP ของ Interface Server
      8. Port = กำหนด Port number สำหรับ Interface Client ต้องตรงกับที่ระบุไว้บน Interface Server (ค่า Default คือ 5555)

      9. จากนั้นคลิก OK
