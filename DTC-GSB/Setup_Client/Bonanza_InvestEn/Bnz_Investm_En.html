    การกำหนดค่า Bonanza Investment Environment

      ขั้นตอนการกำหนดค่า Bonanza Investment Environment มีดังนี้

          1. Folder Application Local บนเครื่อง client เพื่อใช้เก็บโปรแกรม Application ของระบบและไฟล์การใช้งานระบบของ User โดย User สามารถมีสิทธิ์เข้าถึง Read/Write ได้ ยกตัวอย่างเช่น Drive C:\Bonanza การกำหนดค่าในขั้นตอนนี้จะเรียกว่า Local Path

      โครงสร้าง Folder และโปรแกรมสำหรับใช้งานหลักๆ ของ Local Path มีดังนี้


          2. Folder Application Server จะเป็น Share Folder สำหรับใช้งานร่วมกันของระบบ และมีBonanza Package อยู่ที่ Folder Package ในขั้นตอนนี้จะเรียกว่า Server Path Mapping Network Drive จาก ...\storage\invest_client_home

      โครงสร้าง Folder และโปรแกรมสำหรับใช้งานหลักๆ ของ Server Path มีดังนี้


          3. ทำการสร้าง shortcut บน desktop จากไฟล์ดังนี้ Administration.exe, Operation.exe, EnvSetup.exe ,InvestReg.exe
          (โดยการคลิ๊กขวาบนไฟล์นั้นๆ ➢ Send to ➢ Desktop (Create shortcut))


          (Optional)ในกรณีที่ติดตั้ง Bonanza Package ไว้ใน Local Path ของเครื่อง Client อยู่แล้วอาจไม่จำเป็นต้องสร้าง Shortcut ก็ได้

          (Optional) สามารถสร้าง Shortcut ไว้ใน Local Path ของ User ด้วยเพื่อความสะดวกในการใช้งานกรณีบน Desktop มี Shortcut จำนวนมาก
