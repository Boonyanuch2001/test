ย้อนกลับ

    ขั้นตอนการเตรียมระบบก่อนติดตั้ง

    1. ผู้ใช้จะต้องล็อคอินเป็น Administrator ที่มีสิทธิ์ในการเข้าถึงระบบเซอร์วิสของวินโดวส์
    2. ในการตั้งค่าเขตและพื้นที่สามารถทำได้โดย เลือก Start menu ➢ Control Panel ➢ Region and Language แท็บ Formats ให้ตั้งค่าในช่อง Format เป็น English (United States) ดังรูป


    3. แท็บ Administrative ในช่องของ Language for non-Unicode programs ให้เลือกเป็น Thai (Thailand) จากนั้นกดปุ่ม OK และทำการรีสตาร์ทวินโดวส์ใหม่ ดังรูป


    4. ตรวจสอบสิทธิ์ user ต้องสามารถเข้าถึง Share Folder บน Application Server ได้และสามารถ Read/Write Folder Package ได้เนื่องจากระบบ bonanza ทำงานผ่าน client server ผ่าน file protocol บน Windows ไปยัง Application Server
