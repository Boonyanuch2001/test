    ขั้นตอนการ Restore Database

      ขั้นตอนติดตั้งฐานข้อมูลมีหลายวีธี อยู่ที่การเตรียมฐานข้อมูลที่ Backup ไว้
      จะกล่าวถึงวีธีนำไฟล์จาก Backup นำมาติดตั้งใช้งาน เตรียมไฟล์ที่ทำการติดตั้งฐานข้อมูลลงบน Server
      ยกตัวอย่างเช่น C:\Program Files\Microsoft SQL Server\MSSQL16.MSSQLSERVER\MSSQL\Backup
      Login windows ด้วย user SVCctdadmin
      1. เปิด SSMS เลือกเมนูStart ➢ Microsoft SQL Server Management Studio


      2. เลือก Window Authentication ในช่อง และเลือก Trust server certificate แล้ว Connect ดังรูป


      3. หน้าต่าง SQL Server Management Studio Object Explorer ดังรูป


      4. คลิกขวาที่ Databases จากนั้นเลือก Restore Database… ดังรูป


      5. ในหน้าต่าง Restore Databases เลือก Device แล้วทำการกด ... เพื่อเปิดหน้าต่างเลือกไฟล์ดังรูป


      6. จะมีหน้าต่าง Select Backup device ให้ คลิก Add ดังรูป


      7. หน้าต่าง Locate Backup file ให้เลือก Folder ที่เตรียมไว้แล้วเลือก Backup File ที่ต้องการ Restore แล้วกด OK ดังรูป


      8. กลับมาที่หน้าต่าง Select Backup device แล้วกด OK ดังรูป


      9. กลับมาที่หน้าต่าง Restore Databases ช่อง Destination ให้ตั้งชื่อฐานข้อมูล แล้วกด OK ดังรูป


      10. รอ Progress 100% แล้วกด OK ดังรูป


      11. ที่ Object Explorer ในหน้า SQL Server Management Studio จะได้ฐานข้อมูลตามที่ตั้งชื่อไว้ดังรูป
