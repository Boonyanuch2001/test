    การ config certificate

      1. Copy certificate ที่ธนาคาร generate ให้ ctdPrinterService.p12 ไว้ที่ $CTD_PASSBOOK_HOME/cert

      2. เปิด file $CTD_PASSBOOK_HOME\config\application.yaml ตรวจสอบ property server.ssl.keystore

      3. server.ssl.key-store-password ให้ถูกต้อง จากนั้นบันทึก file
